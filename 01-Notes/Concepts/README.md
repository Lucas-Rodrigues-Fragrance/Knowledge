# Concepts

Notes permanentes atomiques : **un concept = une note**, formulé avec vos propres
mots et relié aux autres par des liens `[[...]]`.

Exemples pertinents pour le projet : *gouvernementalité algorithmique*,
*affordance*, *médiation technique*, *capitalisme de surveillance*,
*sociotechnique*, *agentivité distribuée*…

Modèle : `templates/note-permanente.md`.
